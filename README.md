# Unified Mentor Internship Projects

A collection of four web development projects showcasing modern UI/UX design, responsive layouts, and advanced JavaScript functionality.

## Projects Overview

### 1. SkyCast - Weather Dashboard
A feature-rich weather application with real-time data, interactive maps, and beautiful visualizations.

**Key Features:**
- Live weather data from OpenWeatherMap API
- Air Quality Index (AQI) monitoring
- 5-day extended forecast with hourly graphs
- Interactive weather maps (Clouds, Precipitation, Temperature)
- Smart clothing & activity advisor
- Dynamic temperature-based UI themes
- Favorites system with localStorage

**Tech Stack:** HTML5, CSS3, JavaScript ES6+, Chart.js, Leaflet.js, OpenWeatherMap API

Modern Tic-Tac-Toe with AI opponents, multiple themes, and tournament mode.

- Move history with undo functionality

##  Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- For Weather Dashboard: OpenWeatherMap API key ([Get one free](https://openweathermap.org/api))

### Running the Projects

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/internship-projects.git
   cd internship-projects
   ```

2. **Navigate to any project**
   ```bash
   cd project-name
   ```

3. **Open in browser**
   - Simply open `index.html` in your browser
   - Or use a local server:
     ```bash
     # Python 3
     python -m http.server 8000
     
     # Node.js
     npx serve
     ```

4. **For Weather Dashboard only:**
   - Get your API key from OpenWeatherMap
   - Open `script.js` and replace the placeholder:
     ```javascript
     const apiKey = 'YOUR_API_KEY_HERE';
     ```
