# Interactive Web Quiz

A modern, feature-rich quiz application built with HTML, CSS, and JavaScript for testing web development knowledge.

## Features

### Quiz Modes
- Normal Mode: Standard quiz experience with timed questions
- Confidence Mode: Rate your confidence level for each answer to earn bonus points or penalties
- Stress-Free Mode: No time limits, answer at your own pace

### Difficulty Levels
- Easy: 20 seconds per question with fundamental concepts
- Medium: 15 seconds per question with intermediate topics
- Hard: 10 seconds per question with advanced concepts

### Scoring System
- Base points for correct answers
- Streak multiplier: Build consecutive correct answers to multiply your score (up to 3x)
- Boss questions: Every 5th question awards double points
- Confidence bonuses: Earn extra points in Confidence Mode for high-confidence correct answers

### Power-ups
- Hints: Get clues for difficult questions (2 available per quiz)
- Skip: Skip a question you're unsure about (1 available per quiz)
- Time Boost: Add 10 seconds to the current question timer (1 available per quiz)

### Performance Analytics
- Average time per question
- Fastest and slowest response times
- Best streak achieved
- Topic-wise performance breakdown
- Question difficulty heatmap
- Answer change tracking

### User Experience
- Real-time progress tracking
- Visual feedback for correct and incorrect answers
- Detailed explanations for each question
- Keyboard shortcuts for faster navigation (1-4 for options, Enter to submit)
- Animated UI elements and transitions
- Responsive design for all screen sizes

### Leaderboard
- High score tracking
- Local storage for persistent scores
- Display of top 10 scores
- Score history with dates

### Review System
- Full answer review after quiz completion
- Shows your answers, correct answers, and explanations
- Time taken for each question

## Technical Details

### Files
- index.html: Main structure and layout
- script.js: Quiz logic, scoring, and interactions
- style.css: Styling and animations

### Browser Support
Works on all modern browsers that support:
- ES6 JavaScript
- CSS Grid and Flexbox
- Local Storage API

## How to Use

1. Open index.html in a web browser
2. Select your preferred difficulty level
3. Choose a quiz mode
4. Click "Start Quiz" to begin
5. Answer questions within the time limit
6. Use power-ups strategically when needed
7. Review your performance and answers after completion
8. Try again to beat your high score

## Quiz Topics

The quiz covers three main areas of web development:
- HTML: Structure and semantic markup
- CSS: Styling, layout, and design
- JavaScript: Programming concepts and DOM manipulation

## Installation

No installation required. Simply download the files and open index.html in your web browser.
